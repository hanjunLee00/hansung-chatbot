## 📋 프로젝트 개요

- **프로젝트 이름**: 학교 챗봇
- **주요 기능**:
  - 이미지 기반 공지 뿐만 아니라 모든 공지를 학습하여 대답
  - 자주 묻는 질문(FAQ) 탭 구성 (장학금, 학식, 캠퍼스맵, 시설예약)
  - 최근 공지사항 및 학과별 공지 추천
- **목적**: 학교 내 정보 접근성을 향상시킴

---

## 🛠️ 기술 스택

- **프론트엔드**: CSS, Streamlit 라이브러리
- **챗봇 엔진**: OpenAI API
- **데이터베이스**: MySQL, Pinecone Vector DB
- **배포 플랫폼**: AWS
